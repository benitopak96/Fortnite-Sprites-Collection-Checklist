# 🌌 Fortnite Sprite Trading Checklist

An elegant, mobile-friendly checklist web app designed to track your Fortnite Sprite collection and variants. Easily see your completion progress, toggle your inventory, and customize your trading username.

🔗 **Live Link:** [View My Sprite Checklist](https://benitopak96.github.io/Fortnite-Sprites-Collection-Checklist/)

---

## ✨ Features

* **Interactive Toggles:** Quickly tap any sprite variant tile to toggle between **Have it (✓)** and **Need it (✕)**.
* **Custom Epic Username:** Edit your username dynamically at the top of the page, which instantly updates your Call to Action (CTA) at the bottom for quick sharing.
* **Progress Tracking:** Visual gradient progress bar calculates your overall collection completion percentage automatically.
* **Auto-Save:** Powered by `localStorage`—your custom checklist data and username persist natively inside your web browser. No log-ins required.
* **Reset Functionality:** Want to clean the slate or start fresh? Reset back to default values with a single click.

---

## 🛠️ Built With

* **HTML5** - Structure & semantics
* **CSS3** - Modern custom properties (variables), UI styling, and responsive layout grids
* **Vanilla JavaScript** - State management, browser persistence, and dynamic rendering

## 🚀 How to Customize or Use Locally

If you want to tweak the code or run it locally:

1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
3. If you want to change the default starting inventory or change the trackable sprites, update the `SPRITES` array or `DEFAULT_STATE` object inside the `<script>` tags.
